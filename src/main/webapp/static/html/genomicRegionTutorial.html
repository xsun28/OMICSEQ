Genomic Region Search (currently under maintenance)
1. Genomic Region Search Interface

As the figure above shows, genomic region search can be activated by clicking on the "Genomic Region" button on the search bar. A user can search hg19 database for a genomic region from a prespecified list, e.g., promoters, enhancers.
