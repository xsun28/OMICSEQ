Multivariant Search
1. Multivariant Search Interface

As the figure above shows, multivariant search can be activated by clicking on the "Multivariant" button on the search bar. A group of ?????? genomic variants can be searched on hg19 database. It requires a description of the genes as the input parameter. Advanced parameters can be specified by clicking the "Setting" button which includes:

  • Experiments Types: Datasets under search can be filtered by setting a filter on experiments types such as ChIP-seq TF, Dnase-seq, GWAS, etc.

  • Data Source: Datasets can also be filtered by setting a filter on datasoruces such as ArrayExpress, TCGA, GEA, etc.
